## 一、项目概述

### 核心任务
1. **数据获取**：球队历史比赛数据、球员统计、赛况信息
2. **特征工程**：构造近期状态、主客场、命中率等特征，分析其与比赛结果的关系
3. **建模任务**：
   - 胜负预测（分类）
   - 分差预测（回归）
   - 对比不同特征组合的表现
4. **可视化输出**：球队能力雷达图、特征重要性图等

### 技术选型
| 类别 | 技术方案 |
|------|----------|
| 主建模 | 监督学习（随机森林/XGBoost/逻辑回归对比实验） |
| 辅助建模 | 无监督学习（K-means聚类/PCA降维） |
| 深度学习 | 不推荐（数据量不足、可解释性差） |
| 技术栈 | Python + Pandas + Scikit-learn + XGBoost + Plotly |
| 硬件需求 | CPU即可，无需GPU |

### 参考项目
- **GitHub**：https://github.com/saccofrancesco/deepshot
- **技术栈**：NiceGUI + XGBoost + Pandas + Basketball Reference
- **借鉴点**：前端界面、数据爬取思路
- **改进点**：多模型对比、加回归任务、特征工程重做
